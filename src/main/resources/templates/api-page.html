API Endpoint

Use the following API endpoint to recognize license plates:

POST /api/recognize

Headers

Content-Type: multipart/form-data

Parameters

apiKey: Your API Key (string)
file: The image file to be uploaded (multipart/form-data)

Example Request

curl -X POST \
  http://your-domain.com/api/recognize \
  -H 'Content-Type: multipart/form-data' \
  -F 'apiKey=your_api_key' \
  -F 'file=@path_to_your_image'
